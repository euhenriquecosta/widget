🚀 Opções de Acionamento do Widget

Widget não inicializado. Escolha uma opção de acionamento abaixo.

🎯 Abertura Imediata

O widget abre automaticamente assim que a página carrega.

trigger: 'immediate'

⏰ Abertura com Delay

O widget abre após 3 segundos de carregamento da página.

trigger: 'delayed', autoOpenDelay: 3000

📜 Abertura por Scroll

O widget abre quando você rola 50% da página.

trigger: 'scroll', scrollThreshold: 50

🚪 Exit Intent

O widget abre quando o mouse sai da janela (tentativa de sair).

trigger: 'exit-intent'

🔘 Abertura por Botão

O widget abre apenas quando você clica no botão flutuante.

trigger: 'button'

📜 Área de Scroll para Teste

Role para baixo para testar o trigger de scroll.

O widget abrirá automaticamente quando você atingir 50% da página.

Este conteúdo é apenas para demonstrar a funcionalidade.
