✨ Widget Leadnator - Forma Simplificada

Agora você só precisa passar o flowId e os endpoints da API são construídos automaticamente!

Código usado nesta página:

Leadnator.initWidget({
  flowId: '68b4a0cde64414a59012ca37'
});

O que acontece automaticamente:

  • configEndpoint: https://api.leadnator.com.br/api/v1/flows/public/68b4a0cde64414a59012ca37/widget-design
  • domainEndpoint: https://api.leadnator.com.br/api/v1/flows/domains/68b4a0cde64414a59012ca37

Você ainda pode customizar:

Leadnator.initWidget({
  flowId: '68b4a0cde64414a59012ca37',
  title: 'Meu Chat Personalizado', // Sobrescreve config do servidor
  position: 'bottom-left'            // Sobrescreve config do servidor
});
